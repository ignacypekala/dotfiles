# Treesitter
Treesitter is manged by nvim-treesitter module, which manages parsers for the neovim treesitter module.

## Installing a new language
Add language name to treesitter_langs variable in "init.lua". It is possible to install unsupported languages, in which case you should refer to nvim-treesitter documentation. 
[Supported languages](https://github.com/nvim-treesitter/nvim-treesitter/blob/main/SUPPORTED_LANGUAGES.md)
